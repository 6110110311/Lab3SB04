Django เป็น framework ที่ใช้ในการสร้าง Web Application โดยใช้ภาษา Python ใน 
framework ซึ่งเป็นทั้งตัวที่ช่วยในการเชื่อมต่อ Database ไปจนถึงการ Render ข้อมูลในฝั่งของ
Front End ที่แสดงผลข้อมูลอีกด้วย และเมื่อเราทำการสร้าง Web Application เรียบร้อยแล้ว
ก็ upload ขึ้น github แล้วใช้ git clone ใน pythonanywhere เพื่อใช้ในการ Run Web Application
